# ApplyStack Chrome Extension — Install Guide

## Install in Chrome (takes 60 seconds)

1. Open Chrome and go to: `chrome://extensions`
2. Toggle **Developer mode** ON (top-right switch)
3. Click **Load unpacked**
4. Select this `extension` folder
5. Done — the ApplyStack icon appears in your toolbar

## How to use it

### On LinkedIn / Indeed / Glassdoor
1. Navigate to any job listing page
2. The **⚡ Save to ApplyStack** button appears bottom-right of the page
3. Click it — the ApplyStack side panel opens automatically
4. Sign in with your ApplyStack account
5. Your ATS match score loads instantly
6. Click **Tailor My Resume** to open the full builder pre-loaded with that job

### From the toolbar popup
- Click the ApplyStack icon in your Chrome toolbar
- See detected job on current page
- Quick links to Resume Builder, Interview Prep, Salary Guide

## One-time setup
- After installing, click the extension icon and sign in with your applystack.ai account
- If you're using a custom domain or self-hosted server, go to Settings in the popup to update the backend URL

## Supported job boards
- LinkedIn Jobs (`linkedin.com/jobs`)
- Indeed (`indeed.com`)
- Glassdoor (`glassdoor.com`)

More job boards (Lever, Greenhouse, Workday, etc.) coming soon.
